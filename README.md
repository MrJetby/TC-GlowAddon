This addon adds Team glowing for [TreexClans](https://github.com/JetbyMC/TreexClansV3) plugin

[Download addon here](https://modrinth.com/plugin/clanglow)

![Replace this with a description](https://cdn.modrinth.com/data/cached_images/193ac0dab4af8f18ba62aeb75fc9de19bf02b9bb.png)

PacketEvents required

/clan glow - Toggles the glowing effect


## How to Create a GUI
There are two models available: `choose` and `setcolor`.

To open a menu, use the following action:
```txt
[open] gui_id
````
